# ascl1_competence_analysis
RNA-seq, ChIP-seq, ATAC-seq data analysis for Lundie-Brown et al., 2026 manuscript

Raw sequencing files (fastq) and processed files (counts matrices for RNA-seq and bigwigs/BED files for ChIP-seq and ATAC-seq) are deposited on GEO and will be available after publication. 

Note: Claude was used to generate a script, setup_ascl1_analysis.sh, which can be executed to prepare a working environment consistent with this repository and analysis scripts to improve reproducibility
